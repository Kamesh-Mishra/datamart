Discovery:
    finds dataset
    downloads it to temp storage
    stores doc in ES
    tell coordinator

Indexer:
    register with coordinator (with name, allowing for replicas)
    gets pings from coordinator
    gets new dataset notifications from coordinator
    if not interested:
        say "done"
    else:
        say "interested"
        do work
        store metadata doc in ES
        say "done"

Coordinator:
    keeps track of indexers
        pings them, remove if offline
    keeps track of datasets
        remove if no one is interested (anymore)
    gets indexer registration
    gets discovery notifications
